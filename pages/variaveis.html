Vamos exemplificar sobre a Memória do Computador
[ Espaço Variável 1 ] [ Espaço Variável 2 ] [ Variável Resultado ]
Então o Espaço pode ser ocupada de forma variável
No Exemplo o Usuário Digita:
2 para o espaço [ Variável 1 ]
3 para o espaço [ Variável 2 ]
e no espaço [ Variável Resultado ] fica reservado para o resultado
Logo Variável 1 ->[ 2 ] + Variável 2 ->[ 3 ] = Variável Resultado->[ 5 ]
